List all sub-pages with featured image. Each page has it's own visual frame
occupying full width of the content area.

Configuration:
* `params.listframes.showsummary`: `true/false` - show or hide summary for each post

Example:

```yaml
---
title: List Frames
layout: list-frames
params:
  listframes:
    showsummary: true
---
```
